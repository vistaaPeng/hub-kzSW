# 症状指南：头痛

## 头痛的常见类型

### 紧张性头痛
- 双侧头部压迫感、紧箍感
- 轻中度，不影响日常活动
- 与压力、疲劳、颈肩肌肉紧张相关
- 处理：休息、按摩、非处方止痛药

### 偏头痛
- 单侧搏动性疼痛，中重度
- 伴恶心、畏光畏声
- 可有视觉先兆
- 处理：暗室休息，曲普坦类或 NSAIDs

### 丛集性头痛
- 单侧眼眶周围剧烈疼痛
- 每天发作数次，持续 15 分钟至 3 小时
- 伴同侧流泪、鼻塞
- 需专科治疗

## 危险头痛信号（需立即就医）

以下情况可能提示严重疾病，**不要自行处理，立即就医**：

1. **突发雷击样头痛**：数秒内达到最剧烈程度 → 警惕蛛网膜下腔出血
2. **伴发热 + 颈项强直**：脖子僵硬 → 警惕脑膜炎
3. **伴肢体无力、言语不清、视物模糊** → 警惕脑卒中
4. **头痛性质突然改变**（与以往不同）
5. **50 岁后首次出现严重头痛**
6. **头痛伴意识障碍、抽搐**
7. **癌症或免疫缺陷患者新发头痛**
8. **咳嗽、用力时头痛剧烈加重**

## 头痛问诊要点

就医时向医生说明：
- 疼痛部位、性质（胀痛/刺痛/搏动性）
- 持续时间、发作频率
- 诱发和缓解因素
- 伴随症状（发热、呕吐、视力变化）
- 用药史和既往头痛史
