# 高血压

## 概述

高血压是指在未使用降压药的情况下，非同日 3 次测量，收缩压 ≥ 140 mmHg 和/或舒张压 ≥ 90 mmHg。是心脑血管疾病最重要的危险因素。

## 主要症状

高血压早期常无明显症状，被称为"沉默的杀手"。部分患者可出现：

- 头痛（尤其枕部胀痛），晨起明显
- 头晕、耳鸣
- 心悸、胸闷
- 视力模糊
- 鼻出血

## 诊断标准

| 分类 | 收缩压 (mmHg) | 舒张压 (mmHg) |
|------|--------------|--------------|
| 正常血压 | < 120 | < 80 |
| 正常高值 | 120-139 | 80-89 |
| 高血压 1 级 | 140-159 | 90-99 |
| 高血压 2 级 | 160-179 | 100-109 |
| 高血压 3 级 | ≥ 180 | ≥ 110 |

## 危险因素

- 遗传因素、高盐饮食、肥胖、缺乏运动
- 长期精神紧张、过量饮酒、吸烟
- 年龄增长、睡眠呼吸暂停

## 治疗原则

- 生活方式干预：低盐饮食（每日 < 5g）、减重、规律运动、戒烟限酒
- 药物治疗：需医生评估后选择，常用类包括 ACEI、ARB、钙通道阻滞剂、利尿剂等
- 目标：一般患者 < 140/90 mmHg，合并糖尿病或肾病者 < 130/80 mmHg

## 并发症

脑卒中、冠心病、心力衰竭、肾功能损害、视网膜病变。
