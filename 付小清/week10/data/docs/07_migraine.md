# 偏头痛

## 概述

偏头痛是一种常见的原发性头痛，多为反复发作的单侧搏动性头痛，可伴恶心、呕吐、畏光、畏声。女性患病率约为男性的 3 倍。

## 主要症状

- 单侧头痛（约 60%），也可双侧
- 搏动性、跳痛性质
- 中度至重度疼痛
- 活动可加重头痛
- 伴恶心、呕吐
- 畏光（怕光）、畏声（怕吵）
- 部分患者有先兆：视觉闪光、视野缺损、感觉异常，持续 5-60 分钟

## 诱发因素

- 睡眠不足或过多、饥饿、压力、情绪波动
- 特定食物：奶酪、巧克力、红酒
- 月经周期（女性）
- 强光、噪音、气味刺激

## 诊断要点

- 主要依靠病史，需排除继发性头痛
- 符合 ICHD-3 诊断标准：至少 5 次发作，每次 4-72 小时
- 严重或突发头痛需做头颅 CT/MRI 排除器质性病变

## 治疗

### 急性期

- 轻中度：布洛芬、对乙酰氨基酚
- 中重度：曲普坦类药物（如舒马曲普坦）
- 尽早用药，发作初期效果最佳

### 预防性治疗（频繁发作 ≥ 4 次/月）

- β 受体阻滞剂（普萘洛尔）、抗癫痫药（丙戊酸钠、托吡酯）
- 钙通道阻滞剂（氟桂利嗪）
- 记录头痛日记，识别并避免诱因
