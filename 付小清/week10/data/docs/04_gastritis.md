# 胃炎

## 概述

胃炎是胃黏膜的炎症，分为急性和慢性。急性胃炎起病急，慢性胃炎病程长、反复发作。常见诱因包括幽门螺杆菌感染、非甾体抗炎药、酒精、应激等。

## 主要症状

- 上腹部疼痛或不适，餐后加重或缓解（因人而异）
- 腹胀、嗳气、反酸
- 恶心、呕吐
- 食欲减退
- 慢性胃炎可无明显症状，仅表现为消化不良

## 诊断要点

- 胃镜检查是诊断金标准，可观察黏膜充血、水肿、糜烂等
- 幽门螺杆菌检测（呼气试验、粪便抗原、胃镜活检）
- 需与消化性溃疡、胃癌、功能性消化不良鉴别

## 治疗原则

- 去除诱因：戒烟酒、避免刺激性食物、停用损伤胃黏膜的药物
- 幽门螺杆菌阳性者：四联根除方案（质子泵抑制剂 + 两种抗生素 + 铋剂）
- 抑酸治疗：质子泵抑制剂（奥美拉唑等）
- 胃黏膜保护剂

## 饮食建议

- 规律进食，少食多餐
- 避免辛辣、过酸、过烫食物
- 避免空腹饮酒和浓茶咖啡

## 何时就医

- 呕血、黑便
- 持续剧烈腹痛
- 体重明显下降
- 吞咽困难
