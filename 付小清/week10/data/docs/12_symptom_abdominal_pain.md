# 症状指南：腹痛

## 腹痛部位与可能病因

### 上腹部

- **胃痛/胃炎/消化性溃疡**：上腹正中或偏左，餐后加重或空腹时加重
- **胆囊炎/胆石症**：右上腹疼痛，可放射至右肩，高脂饮食后诱发
- **胰腺炎**：上腹持续性剧痛，向背部放射，伴呕吐

### 脐周

- **肠炎**：脐周隐痛或绞痛，伴腹泻
- **肠梗阻**：阵发性绞痛，伴呕吐、停止排气排便

### 下腹部

- **阑尾炎**：初期脐周痛，后转移至右下腹，McBurney 点压痛
- **泌尿系结石**：腰腹部绞痛，向会阴放射，伴血尿
- **妇科疾病**：痛经、宫外孕（急症）、卵巢囊肿扭转

## 急腹症信号（立即就医）

以下情况提示可能需要急诊手术或紧急处理：

1. 剧烈腹痛持续不缓解
2. 腹部僵硬（板状腹）
3. 伴高热、休克表现（面色苍白、出冷汗、血压下降）
4. 呕血、便血、黑便
5. 腹痛伴停止排气排便（肠梗阻）
6. 育龄女性腹痛伴停经（排除宫外孕）

## 常见功能性腹痛

- **肠易激综合征（IBS）**：腹痛伴排便习惯改变，排便后缓解
- 与压力、饮食相关，无器质性病变

## 家庭观察要点

- 记录疼痛部位、性质、时间
- 观察有无发热、呕吐、腹泻
- 暂禁食禁水（怀疑急腹症时）
- 不要自行服用强效止痛药（可能掩盖病情）

## 就诊科室

- 普通腹痛 → 消化内科
- 右下腹痛 → 普外科（排除阑尾炎）
- 腰痛血尿 → 泌尿外科
- 女性下腹痛 → 妇科
