# 症状指南：咳嗽

## 咳嗽的分类

### 按时间分类

| 类型 | 持续时间 |
|------|---------|
| 急性咳嗽 | < 3 周 |
| 亚急性咳嗽 | 3-8 周 |
| 慢性咳嗽 | > 8 周 |

## 可能病因分析

### 急性咳嗽（< 3 周）

- **感冒/流感**：伴鼻塞、咽痛、发热，自限性
- **急性支气管炎**：咳嗽伴咳痰，多由病毒感染
- **肺炎**：咳嗽 + 高热 + 胸痛 + 呼吸困难

### 亚急性咳嗽（3-8 周）

- 感染后咳嗽：感冒痊愈后咳嗽持续，逐渐自行缓解
- 需排除哮喘、鼻后滴漏

### 慢性咳嗽（> 8 周）常见四大原因

1. **咳嗽变异性哮喘**：干咳为主，夜间和清晨加重
2. **鼻后滴漏综合征**：鼻炎、鼻窦炎导致分泌物倒流刺激咽喉
3. **胃食管反流**：平卧时加重，伴反酸烧心
4. **嗜酸性粒细胞性支气管炎**

## 痰液性状提示

- **白色稀痰**：病毒感染、慢性支气管炎
- **黄色脓痰**：细菌感染
- **铁锈色痰**：肺炎链球菌肺炎
- **粉红色泡沫痰**：急性左心衰竭（急症）
- **血丝痰**：支气管扩张、肺结核、肺癌（需就医）

## 家庭护理

- 多饮水，保持气道湿润
- 蜂蜜（1 岁以上）可缓解咳嗽
- 避免烟雾、粉尘刺激
- 慎用止咳药，痰多时不宜强力镇咳

## 何时就医

- 咳嗽超过 3 周不缓解
- 咳血或痰中带血
- 伴呼吸困难、胸痛、高热
- 夜间咳嗽严重影响睡眠
- 消瘦、盗汗
