# 感冒（急性上呼吸道感染）

## 概述

感冒是由病毒感染引起的上呼吸道炎症，是最常见的急性呼吸道疾病。常见病毒包括鼻病毒、冠状病毒、流感病毒等。全年均可发病，冬春季高发。

## 主要症状

- 鼻塞、流涕、打喷嚏
- 咽痛、咽干
- 咳嗽，可为干咳或有少量痰
- 低热或中等程度发热（通常 < 39°C）
- 头痛、全身酸痛、乏力
- 症状通常 7-10 天自行缓解

## 诊断要点

1. 急性起病，以上呼吸道症状为主
2. 体检可见咽部充血，扁桃体可无肿大
3. 血常规通常正常或白细胞偏低，淋巴细胞比例可升高
4. 需与流感、过敏性鼻炎、急性咽炎等鉴别

## 治疗与护理

- 休息、多饮水、保持室内空气流通
- 对症治疗：退热（对乙酰氨基酚、布洛芬）、缓解鼻塞（生理盐水冲洗）
- 不推荐常规使用抗生素（病毒性感染）
- 避免劳累，注意保暖

## 何时就医

- 高热持续超过 3 天（> 39°C）
- 呼吸困难、胸痛
- 症状持续超过 10 天无好转
- 婴幼儿、老年人、孕妇、免疫力低下者
