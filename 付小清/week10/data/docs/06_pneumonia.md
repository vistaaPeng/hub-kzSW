# 肺炎

## 概述

肺炎是肺泡和肺间质的炎症，可由细菌、病毒、支原体等引起。社区获得性肺炎（CAP）是最常见类型。

## 主要症状

- 发热：多为高热（39-40°C），伴寒战
- 咳嗽：初期干咳，后咳脓痰（铁锈色痰提示肺炎链球菌）
- 胸痛：深呼吸或咳嗽时加重
- 呼吸困难、气促
- 全身乏力、食欲减退

## 诊断要点

- 胸部 X 线或 CT：肺部片状浸润影
- 血常规：白细胞升高，中性粒细胞比例增高（细菌性）
- 痰培养、血培养明确病原菌
- 血气分析评估缺氧程度

## 常见病原体

- 细菌：肺炎链球菌、金黄色葡萄球菌、肺炎克雷伯菌
- 病毒：流感病毒、新冠病毒
- 非典型：支原体、衣原体（多见于青少年）

## 治疗原则

- 细菌性肺炎：抗生素治疗，根据病原学选择
- 病毒性肺炎：对症支持，重症可用抗病毒药
- 氧疗：血氧饱和度 < 93% 时吸氧
- 止咳化痰、补液、营养支持

## 何时就医

- 高热持续不退
- 呼吸困难加重
- 老年人、儿童、免疫力低下者一旦怀疑肺炎应立即就医
