# 症状指南：发热

## 发热的定义

腋温 > 37.3°C、口温 > 37.5°C、肛温 > 38.0°C 可视为发热。

## 按热度分级

| 分级 | 体温 |
|------|------|
| 低热 | 37.3 - 38.0°C |
| 中等热 | 38.1 - 39.0°C |
| 高热 | 39.1 - 41.0°C |
| 超高热 | > 41.0°C |

## 可能病因分析

### 低热 + 鼻塞流涕咽痛 → 普通感冒
病毒性上呼吸道感染，通常自限，对症处理。

### 高热 + 寒战 + 咳嗽咳痰 → 肺炎
需就医，可能需要抗生素和影像学检查。

### 高热 + 头痛全身酸痛 → 流感
流感病毒感染，抗病毒药（奥司他韦）在 48 小时内使用效果最佳。

### 持续低热 + 盗汗消瘦 → 需排查结核等
长期不明原因低热应就医做系统检查。

### 儿童发热注意事项
- 3 个月以下婴儿发热 ≥ 38°C 应立即就医
- 高热惊厥：体温快速上升时可能发生，需紧急处理

## 家庭处理

- 多饮水，物理降温（温水擦浴）
- 退热药：对乙酰氨基酚或布洛芬（按说明书剂量）
- 观察伴随症状变化

## 必须就医的情况

- 体温 > 39°C 持续 3 天以上
- 伴呼吸困难、意识改变、皮疹
- 婴幼儿、老人、孕妇发热
- 免疫力低下者（化疗、艾滋病等）
